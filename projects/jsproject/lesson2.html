Вирази у JS

Розглянемо вираз у наступній команді:

Чому буде дорівнювати змінна total, якщо price = 10, discount = 20?

var total = price - (price * discount / 100);

Відповідь: 8.

Існують і рядкові вирази:

"Dear " + "Reader" + "," (конкатенація)
"super" + "cali" + youKnowTheRest

Результатом обчислення також може бути істиною чи хибою (true або false). Такі вирази називають булевими (або логічними).

age < 14
cost >= 3.99
animal == "bear" - істина, якщо назва тварини bear

Результатом обчислення виразу буде число, рядок або логічний вираз.

=    -  оператор присвоювання
==  -  оператор перевіряє чи рівні два значення
!=   -  оператор перевіряє що два значення НЕ рівні

Цикл while

В програмах операції доводиться повторно виконувати. Для цього у JS використовуються конструкції while, for, for in, for Each.

Розглянемо приклад:

while (scoops > 0) {
  document.write("Another scoop!");
  scoops = scoops - 1;
}

while - ключове слово

scoops > 0 - умова

В дужках { } - програмний блок

Прийняття рішень в JS (умовний оператор if)

Розглянемо приклад:

if (scoops < 3) {
  alert("Ice cream is running low!");
}

if - ключове слово

scoops < 3 - умова

В дужках { } - програмний блок команди if

Функція alert отримує рядок і виводить його у спливаючому вікні браузера.

Команда if дозволяє послідовно виконувати декілька перевірок:

if (scoops >= 5) {
  alert("Eat faster, the ice cream is going to melt!");
} else if (scoops == 3) {
  alert("Ice cream is running low!");
} else if (scoops == 3) {
  alert("Going twice!");
} else if (scoops == 3) {
  alert("Going once!");
} else if (scoops == 3) {
  alert("Gone!");
} else {
  alert("Still lots of ice cream left, come get it");
}
Функція prompt отримує рядок, виводить його у спливаючому вікні браузера та чекає, поки користувач введе текст, а потім повертає введене значення або null, якщо введення скасоване (Cancel/ESC). Результат роботи функції prompt є типом string, тобто рядком.
var age = prompt("How old are you?");
alert(age);
Якщо потрібен результат типу number, тобто число, необхідно використати +prompt замість prompt: 
var age = +prompt("How old are you?");
alert(age);

Закріплення знань

1) Яким буде результат, якщо temp дорівнює 10?

(9 / 5) * temp + 32

Правильна відповідь: 50.

2) Істиною чи хибою буде рівняння, якщо змінна color містить значення "pink"?

color == "orange"

Правильна відповідь: хибою.

3) Істиною чи хибою буде рівняння, якщо змінна color містить значення "orange"?

color == "orange"

Правильна відповідь: істиною.

4) Яким буде результат виразу?

1000 + "108"

Правильна відповідь: 1000108.

Написати програми:

5) Знайти суму всіх двозначних чисел.

var sum = 0;

for (var i = 0; i < 100; i++) {
  sum =  sum + i;
}

alert("Сума всіх двозначних чисел = " + sum);

6) Вивести числа які кратні 7 та не перевищують 100 (7, 14, 21... 98).

for (var i = 1; i <= 100; i++) {
  if (i % 7 == 0) {
    alert("Число " + i + " кратне 7");
  }
}

7) Вивести квадрати натуральних чисел, які менші або дорівнюють n.

var n = +prompt("Введіть число n: ");

for (var i = 0; i <= n; i++) {
  alert("Квадрат числа " + i + " = " + i*i);
}

8) Чи існує трикутник зі сторонами a, b, c?

var a = +prompt("Введіть сторону a: ");
var b = +prompt("Введіть сторону b: ");
var c = +prompt("Введіть сторону c: ");
var result = false;

if (a + b > c) {
  if (a + c > b) {
    if (b + c > a) {
    	result = true;
    }
  }
}

if (result) {
  alert("Так, може існувати");
}
else {
  alert("Ні, не може існувати");
}

9) Знайти найбільше з чисел a, b, c.

var a = +prompt("Введіть число a: ");
var b = +prompt("Введіть число b: ");
var c = +prompt("Введіть число c: ");

if (a > c && a > b) {
  alert("Найбільше число з a, b, c - a");
} else if (b > a && b > c) {
  alert("Найбільше число з a, b, c - b");
} else if (c > b && c > a) {
  alert("Найбільше число з a, b, c - c");
} else {
  alert("Немає найбільшого числа");
}

10) Дано час (години і хвилини). Скільки буде через хвилину?

var hours = +prompt("Введіть години: ");
var minutes = +prompt("Введіть хвилини: ");

if ((hours >= 0 && hours < 24) && (minutes >= 0 && minutes < 60)) {
  if (minutes == 59) {
    if (hours < 23) {
      hours = hours + 1;
      minutes = "00";
    } else {
      hours = 0;
      minutes = "00";
    }
  } else {
    minutes = minutes + 1;
    if (minutes < 10) {
      minutes = "0" + minutes;
    }
  }
  if (hours < 10) {
    hours = "0" + hours;
  }
  alert("Буде " + hours + ":" + minutes);
} else {
  alert("Ви ввели неправильний час!");
}
