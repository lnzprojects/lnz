Атрибути

Зробимо текст: "Red Alert" на малюнку червоним тривожним кольором:

  1. Визначимо для класу "redtext" правило CSS, що надає абзацу червоний колір тексту.
  2. Додамо код, який бере елемент абзацу "greenplanet" і додає до нього клас redtext.
Об’єкти element містять метод setAttribute, який може викликатися для завдання атрибутів елементів HTML.
Метод використовується для додавання чи зміни вже існуючого атрибута.
planet.setAttribute("class", "redtext");

Метод отримує два аргументи:

  • ім’я атрибута
  • значення атрибута

Наприклад:

Значення атрибута можна читати методом getAttribute.
var scoop = document.getElementById("raspberry");
var altText = scoop.getAttribute("alt");
console.log("I can not see the image in the console,");
console.log("but i'm told it looks like: " + altText);
Якщо шуканий атрибут не існує в елементі то ми отримаємо null.

Перевірка:

var scoop = document.getElementById("raspberry");
var altText = scoop.getAttribute("alt");

if (altText == null) {
  console.log("No here an alt attribute");
} else {
  console.log("I can not see the image in the console,");
  console.log("but i'm told it looks like: " + altText);
}

Що ще можна робити з моделлю DOM?

  • Отримувати елементи (всі елементи класу "onsale", значення введені користувачем в формі).
  • Створювати і додавати елементи в DOM.
  • Вилучати елементи.
  • Перебирати елементи (дочірні даного, сусідні).

Закріплення знань

Завдання: додати до веб-сторінки зафарбування в червоний колір (просто додати: planet.setAttribute("class", "redtext"); в функцію init + стилізацію для класа redtext.

1. Вивести квадрат вмісту input (9 -> 81).

<!DOCTYPE html>
<html>
<head>
  <title>document</title>
</head>
<body>
  <input placeholder="enter a number" value="" id="input">
  <button onclick="numberQ()">click</button>
  <script type="text/javascript">
    function numberQ() {
      var text = document.getElementById("input").value;
      alert(text*text);
    }
  </script>
</body>
</html>

2. Зробити перевірку чи є значення що водиться числом.

<!DOCTYPE html>
<html>
<head>
  <title>document</title>
</head>
<body>
  <input placeholder="enter a number" value="" id="input">
  <button onclick="isItNumber()">click</button>
  <script type="text/javascript">
    function isItNumber() {
      var text = document.getElementById("input").value;
      if (!isNaN(text)) {
        alert("Введене значення є числом");
      } else {
        alert("Введене значення не є числом");
      }
    }
  </script>
</body>
</html>

3. Зміна зображення при натисканні на кнопку.

<!DOCTYPE html>
<html>
<head>
  <title>document</title>
</head>
<body>
  <img id="image" src="image1.jpg">
  <button onclick="changeImg()">click</button>
  <script type="text/javascript">
    function changeImg() {
      document.getElementById("image").src = "image2.jpg";
    }
  </script>
</body>
</html>

4. Зміна вмісту (-><-) двох input.

<!DOCTYPE html>
<html>
<head>
  <title>document</title>
</head>
<body>
  <input placeholder="enter text" value="" id="input1">
  <input placeholder="enter text" value="" id="input2">
  <button onclick="changeInput()">click</button>
  <script type="text/javascript">
    function changeInput() {
      var text = document.getElementById("input1").value;
      document.getElementById("input1").value = document.getElementById("input2").value;
      document.getElementById("input2").value = text;
    }
  </script>
</body>
</html>

5. Зміна властивості float для input.

<!DOCTYPE html>
<html>
<head>
  <title>document</title>
</head>
<body>
  <span>text</span>
  <input placeholder="enter text" value="" id="input">
  <button onclick="floatInput()">click</button>
  <script type="text/javascript">
    function floatInput() {
      document.getElementById("input").style.float = "right";
    }
  </script>
</body>
</html>
