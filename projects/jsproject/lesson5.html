Логічні оператори

Припустимо, потрібно перевірити чи є товар в наявності і чи продається він (змінні inStock і onSale відповідно)

  • 1 спосіб
    • if (inStock == true) {
      if (onSale == true) {
        alert("buy buy buy!");
      }
    }
  • 2 спосіб
    • if (inStock == true && onSale == true) {
      alert("buy buy buy!");
    }
  • 3 спосіб (скорочений)
    • if (inStock && onSale) {
      alert("buy buy buy!");
    }

Випадкове розміщення кораблів

Для того щоб вірно розмістити корабель нам потрібна початкова позиція, від якої будуть відраховуватися три суміжні клітинки. Отже, початкова клітинка має належати діапазону від 0 до 4.

Якщо у нас є початкова позиція randomLoc (від 0 до 4), то:

var location1 = randomLoc;
var location2 = location1 + 1;
var location3 = location2 + 1;
Math.random() - функція, яка повертає випдкове число діапазону від 0 до 1 (не включаючи 1, наприклад 0,128..).
var randomLoc = Math.random();

Якщо домножимо випадкове число на 5 то отримаємо число з [0;5). Залишилось прибрати дробову частину.

var randomLoc = Math.floor(Math.random() * 5);
Math.floor() - функція, яка округлює число, прибираючи дробову частину.

Є ще одна помилка – якщо тричі ввести одне й те саме число, то корабель потоне.

Напишемо функцію для цього фрагменту коду.

Запам'ятайте!

  • Оголошення функції завжди починається з ключового слова function

Закріплення знань

1) На скільки треба домножити щоб отримати випадкове число від 0 до 100 включно?


Правильна відповідь: 101

Написати програми використовуючи ф-її:

2) Факторіал числа n.

var inputNumber = prompt('Please enter a number: ');
var result = 1;

for(var i = 0; i < inputNumber; i++) {
  result = result * (inputNumber - i);
}

console.log(inputNumber + '! = ' + result);

3) Знайти n в степені m.

var n = prompt('Please enter a number: ');
var m = prompt('Please enter a number: ');
var result = n;

for (var i = 2; i <= m; i++) {
  result = result * n;
}

console.log(n + ' в степені ' + m + ' = ' + result);

4) Вивести n-ий член послідовності Фібоначчі.

var n = prompt('Please enter a number: ');
var a = 1;
var b = 1;
var c = 1;

for (var i = 3; i <= n; i++) {
  a = b;
  b = c;
  c = a + b;
}

console.log(n + 'ий член послідовності Фібоначчі = ' + с);
